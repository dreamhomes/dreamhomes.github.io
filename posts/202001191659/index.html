Windows 高效软件推荐

发表于2020-01-19|更新于2022-05-26|工具篇
|字数总计:1.2k|阅读时长:3分钟|阅读量:|评论数:

本文对2019年所使用的windows软件工具进行总结和推荐。

桌面美化


此图片为本人目前桌面截图，以简洁实用为标准。主要涉及到以下软件：

每日壁纸精选

微软商店 每日壁纸精选

此应用可以自动下载必应壁纸来自动更新电脑桌面壁纸。应用占用开机内存较小，壁纸质量高而且免费，因此选择这款应用。

TranslucentTB

github TranslucentTB

此应用可以将菜单栏虚化或者透明化，占用内存较小，个人感觉使桌面更加美观。也可以直接在微软商店下载。效果如下所示：

文件及其快捷方法


此处可根据自身的需求和习惯进行定制，个人习惯如下：

  1. 桌面上仅保存一个文件夹work;里面保存待处理的文件和一些快捷方式，可以使桌面看起来更加整洁。
  2. 任务栏中固定一些使用频率较高的软件，减少搜索时间。

系统效率工具

Snipaste 截图工具

官网地址

目前发现最好用的截图工具，可以在屏幕上贴上多个图片+自定义截图快捷键，在阅读、撰写文献、对比图片中非常实用。强烈推荐！

火绒安全软件

官网地址

目前使用存在感最低的软件，因为这款软件中不绑定软件、没有广告弹窗等等多个优点。强烈推荐其中的弹窗拦截功能，使用之后打开电脑再也不会看见满屏幕的弹窗广告了。

Bandizip 解压工具

官网地址

无广告、开源的文件解压工具，支持文件预览，这三个功能已经完美符合我的需求。

CCleaner清理工具

官网地址

轻量系统清理工具，可以全面清理系统缓存垃圾和注册表。目前最实用的清理工具。完整版可以@我哦！

pandownload下载工具

官网地址

百度云盘不限速下载工具。有时没有下载速度提升不大，可以时刻关注软件的更新情况。当软件效果不好时可以使用另一替代品 baidupcs-web，这款软件加速不稳定但是普遍比网盘快。

印象笔记+有道云笔记

目前市场上的笔记软件对比可参考: 笔记对比。由于企业中使用印象笔记较多，个人感觉有道云笔记编辑方便，因此同时使用这两款软件。

Xmind 思维导图

官网地址

此款思维导图软件界面简洁，模板齐全。需要全功能版可以@我哦！

Recuva 文件恢复

官网地址

ccleaner同企业产品，误删某文件时可以进行恢复并且可以恢复多种类型文件。

浏览器

google 官网地址

目前体验最好的浏览器，没有之一。强悍的插件工具，perfect！！！

windows 便笺

windows自带的便笺工具，没有的可以在微软商店下载。本人将其固定在任务栏中用于记录每天的待办事项。

Potplayer 视频播放

官网地址

最强的视频播放工具，软件小巧而且支持多种视频格式，目前用的最顺手的视频播放工具。

其它

  1. 百度网盘
  2. 迅雷下载
  3. 网易云音乐
  4. photoshop
  5. …

生产力工具

Typora md文本编辑

官网地址

目前体验最好的markdown编辑工具，即写即视感特别好。

mathpix 公式转换工具

官网地址

截图自动转换为Latex代码。目前免费版对于转换次数有限制了，所以可以注册多个账号！

TrafficMonitor 系统监控

官网地址

轻量的系统监控工具，用于查看网速和系统状态。

pycharm + vscode + sublime

对于本人python开发者，这三款工具是必备的，各有各的优点。对于pycharm，一般用于大型项目开发，对于vscode一般用于小型项目和远程开发，对于sublime一般用于替代文本编辑器。

TeXStudio

官网地址

目前体验感最好的Latex编辑器，论文撰写必备工具。

Visio 流程图

微软的这款流程图工具与draw.io效果都不多，对于不能使用的同学使用visio工具。

Gridea博客工具

官网地址

以前都是使用Hexo搭建博客，发现这款工具后即转用可视化的静态博客工具。开发者也在不断完善其中的功能，支持！！！

Shadowsocks网络代理

目前用过最好的网络代理软件SS，需要自己购买服务。

以上是本人2019年感觉较为实用的软件，特此总结推荐。如果博友有更好的工具可以在评论中推荐哦！明年可以再进行总结！😊😊😊🙌🙌🙌
