MacOS |  优秀的软件工具总结篇

发表于2021-01-25|更新于2022-05-26|工具篇
|字数总计:504|阅读时长:1分钟|阅读量:|评论数:

推荐个人高效的 macOS 软件，随所遇而更新…

系统工具

  • Chrome：多端信息和插件同步浏览器；
  • Cheatsheet：显示软件快捷键；
  • Rectangle/magnet：快捷分屏工具；
  • Paste：粘贴板工具；
  • Pap.er：壁纸工具，体验感较差
  • Unsplash：锁屏壁纸。
  • Alfred：快速查找及其入口；
  • 滴答清单：每日任务提醒；
  • F.lux：屏幕亮度调节；
  • ClashX：系统代理软件；https://github.com/yichengchen/clashX
  • 海豚湾：稳定的翻墙账号；https://hitun.io/
  • iShot：系统截图贴图工具；
  • iStat Menus：系统监控工具；
  • 腾讯柠檬：系统清理工具，附带基础系统监控功能，替代 iStat Menus。
  • Bartender：菜单栏图标整理工具；
  • The Unarchiver：文件解压缩；
  • OmniGraffle：强大的流程图绘画软件；
  • Parallels Desktop：强悍的虚拟机软件；
  • PDF Expert：PDF 文档编辑工具；
  • Free Download Manager：百度网盘下载工具需结合油猴脚本；
  • Neat Download Manager：浏览器下载插件；
  • 搜狗输入法：注意设置中文和英文、数字间添加空格；
  • syncthing：开源的多端文件同步工具。
  • 超级右键：右键添加多种功能。
  • EasyNewFile：右键菜单功能扩充。
  • Just Focus：定时锁屏休息。

写作工具

  • Typora： markdown 协作软件;
  • PicGo + Github 图床；
    • 插件：Compress + Watermark；
  • Mathpix Snipping Tool：公式转 Latex 代码；
  • Notion：个人记录；
  • Xmind：思维导图；
  • Zotero: 文献管理工具

开发工具

  • Vscode + PyCharm：Python 开发必备，VSCode 插件配置可以参考另一篇文章
  • IntelliJ IDEA： Scala、Java 开发必备；
  • Dash：编程文档搜索查看，可结合Alfred；
  • 文档撰写工具
    • docsify https://docsify.js.org/#/
    • gitbook https://www.gitbook.com/
    • vuepress https://vuepress.vuejs.org/
  • iTerm + oh-my-zsh
    • Powerleve 10k 主题
  • Tmux + oh-my-tmux https://github.com/gpakosz/.tmux
  • ELK (Elastic + LogStash + Kibana) 数据处理与接入；

休闲工具

  • 网易云：听听歌
  • 微信：众所周知
  • IINA：视频播放软件
  • Downie：视频下载

Chrome

  • 浏览器插件

  • 油猴脚本列表，其它下载链接：https://www.baiduyun.wiki/

软件下载
